Yahoo Finance Pipeline

Este sitio resume de manera visual el flujo educativo para descargar, procesar y analizar precios de activos con Python.

Datos generales

El proyecto usa Python 3.11, yfinance para las descargas y scikit-learn para los modelos. El código está pensado para ejecutarse de forma automática mediante GitHub Actions.

Visualizaciones diarias

Las gráficas provienen del módulo visualization y se actualizan de forma automática.

Acerca del proyecto

El repositorio contiene scripts para cada fase del proceso:

  • Descarga y limpieza de datos financieros.
  • Ingeniería de variables y selección de características.
  • Entrenamiento y evaluación de modelos de pronóstico.
  • Generación de reportes y notificación de resultados.
